Research Interests

I love systems programming. I am very familiar with the Linux programming interface and system call API. Also, I am a big fan and avid user of the Rust programming language.

My doctoral research uses low-level systems techniques and deterministic parallelism to provide systems and solutions for various software engineering problems. For example, we created Reproducible Containers (DetTrace), a container abstraction for Linux which guarantees both determinism and reproducibility for any unmodified Linux program run through it. This means instead of flaky builds, we can have automatic reproducible builds. The open source code can be found here.

My current research uses low-level systems techniques like system call interception and modification and language level techniques like asynchronous Rust futures to provide automatic caching of computation at the process level through a system we call Process Cache.

During the summer of 2020, I interned with the VMWare Research Group. I worked somewhat outside my normal realm, but still with the Rust programming language, on Distributed Differential Datalog (D3log). D3log is an extension of the Differential Datalog language (built on Datalog, which was specifically designed for incremental computation), which provides automatic distribution of Differential Datalog computations across compute nodes.
