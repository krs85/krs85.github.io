Projects

We created Reproducible Containers (DetTrace), a container abstraction for Linux which guarantees both determinism and reproducibility for any unmodified Linux program run through it. This means instead of flaky builds, we can have automatic reproducible builds. The open source code can be found here. My main contribution to the project was extending the container's sequential scheduler to run in parallel in system-call-free regions of code, bringing the overhead down to just 2% for some of our benchmarks.

My current research combines low-level systems methods with language level techniques like asynchronous Rust futures to provide automatic caching of arbitrary, unmodified Linux programs at the process level through a system we call Process Cache.

During the summer of 2020, I interned with the VMware Research Group. I worked somewhat outside my normal realm, but still with the Rust programming language, on Distributed Differential Datalog (D3log). D3log is a typed Datalog Rust engine built upon a timely dataflow computation model extended to work across machines. Specifically, I adapted the distributed API to allow for incremental, on-the-fly reconfiguration of the nodes within the network, improving fault tolerance.
